How to use this code:

First step: Click on the address bar and type "cmd" (without " "). And press enter. 

2nd step: After the command line appear. Type "python SDKDetector.py -q <Name of test>.
For example: python SDKDetector.py -q Test1.png

Press enter and wait...

*Note. 

- Extracted folder is for storing all elements extracted when we use the code. 
- Test folder is used for testing the SVM algorithm.
- Train folder is used for training the SVM detection. 
- SVMTraining.py is the code for training number digit detection system. 
- SVM_Final.pkl is a file that our SVM training network is saved. 

SVM accuracy: 96%.
